## 🌌 What is Memet?

**Memet** is a full-stack emotionally intelligent AI companion — built for humans who need a safe, warm, and completely non-judgmental space to feel heard.

In a world where people bottle up anxiety, grief, loneliness, and confusion because they don't want to "burden" others — Memet exists as a gentle presence that always has time for you. It doesn't offer hollow advice. It doesn't rush you toward solutions. It **listens first**. It validates what you feel. And then, only when you're ready, it walks with you.

This isn't a chatbot with a purple skin. Memet was **designed from the ground up** with emotional intelligence as the core product — not a feature. Every word in its system prompt, every UI decision, every interaction flow was crafted to make the user feel:

> *"I am not alone in this."*

Built over **a few focused days** by a 2nd-year engineering student — from concept to full-stack live deployment — Memet represents what happens when technical skill meets genuine human empathy.

## 🎨 Frontend — Design & Engineering

Memet's frontend is a **single HTML file** that delivers a full premium web application experience. No React. No build pipeline. No npm. Pure, handcrafted HTML, CSS, and JavaScript — making it fast, portable, and impressively lightweight.

### 🖼️ Visual Identity
- 💜 **Dreamy soft purple/lavender** color system — calm, safe, emotionally intentional
- 🌈 **Animated gradient background** — subtle, breathing, alive
- 🔮 **Three glowing ambient orbs** — creates visual depth and warmth
- 🪟 **Glassmorphism UI** — frosted glass cards and navbar
- 🖋️ **Cormorant Garamond typography** — literary, elegant, emotionally resonant

### ⚙️ Technical Highlights
- 🎯 **120 floating canvas particles** — each drifting independently via `requestAnimationFrame`
- 🖱️ **Custom cursor** with purple glow trail — built purely in JS
- 💫 **Sparkle burst effects** on key interactions
- 📜 **Smooth scroll indicator** on the hero section
- 💬 **Floating animated chat preview card** — lets users see Memet in action before clicking Chat

### 📄 Pages

| Page | Description |
|:-----|:------------|
| 🏠 **Home** | Hero section, animated preview card, call-to-action |
| 💬 **Chat** | Full real-time conversation interface with mood selector |
| 👤 **About** | Project philosophy, founder info, mission statement |

### 💬 Chat Interface Features
- **Mood selector chips** — Sad / Anxious / Frustrated / Numb / Okay / Good
- **Typing indicator** — three bouncing dots while Memet replies
- **Crisis detection** — surfaces Indian mental health helplines (iCall, Vandrevala Foundation, SNEHI) when distress signals are detected
- **Conversational context** — last 12 messages sent with every request for coherence
- **Cold-start UX** — "Waking up, just a moment 💜" message handles Render's free-tier spin-down gracefully

## ⚡ Backend — API Reference

Built with **FastAPI** — one of the fastest Python web frameworks, with automatic OpenAPI/Swagger documentation.

### Endpoints

| Method | Endpoint | Description |
|:-------|:---------|:------------|
| `GET` | `/` | Server info and status |
| `GET` | `/health` | Health check |
| `POST` | `/chat` | Core AI endpoint |

### `POST /chat` — Request
```json
{
  "message": "I feel like no one really understands me",
  "history": [
    { "role": "user", "content": "I've been feeling really off lately" },
    { "role": "assistant", "content": "That sounds exhausting to carry..." }
  ],
  "mood": "Sad"
}
```

### `POST /chat` — Response
```json
{
  "reply": "Feeling unseen by the people around you is one of the loneliest experiences there is. You don't have to explain or justify that — it makes complete sense that it hurts."
}
```

### AI Configuration
```python
model       = "llama-3.3-70b-versatile"
temperature = 0.85    # warm, expressive, human-feeling responses
max_tokens  = 320     # focused replies — not walls of text
context     = last 12 messages sent with every request
```

## 🛠️ Run Locally

### Prerequisites
- Python 3.10+
- Free Groq API key from [console.groq.com](https://console.groq.com)

### Steps

```bash
# 1. Clone the repo
git clone https://github.com/Adityamax0/MEMET.git
cd MEMET

# 2. Create your .env file
echo GROQ_API_KEY=your_gsk_key_here > .env

# 3. Install dependencies
pip install -r requirements.txt

# 4. Start the backend
python -m uvicorn main:app --reload
# → Backend runs at http://localhost:8000
# → API docs at  http://localhost:8000/docs

# 5. Open the frontend
# Just open index.html in your browser
```
